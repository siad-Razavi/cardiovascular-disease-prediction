# Cardiovascular Disease Prediction using Machine Learning

This project presents a machine learning approach for predicting cardiovascular disease using patient health and lifestyle data.

## Objective

The objective of this project is to build a classification model that predicts whether a patient has cardiovascular disease based on medical and lifestyle features.

## Dataset

The dataset contains information such as:
- age
- gender
- height
- weight
- blood pressure
- cholesterol
- glucose
- smoking
- alcohol intake
- physical activity

The target variable is `cardio`, where:
- `0` indicates no cardiovascular disease
- `1` indicates cardiovascular disease

## Methods

The notebook includes:
- data loading
- data cleaning
- exploratory data analysis
- train-test split
- feature scaling
- Logistic Regression model training
- model evaluation

## Result

The model achieved an accuracy of approximately **72.86%**.
